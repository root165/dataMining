# 第4章考察重点：数据探索与描述性统计

**来源依据**：教材第4章目录、期末复习大纲pdf、图片OCR：3A49、917A、5A55、DEB4、4579、DDD4等。

> 说明：本文件只整理 PDF 章节目录、复习大纲、期末复习大纲 PDF 与图片 OCR 中出现的考试重点；未在材料中出现的知识点不扩展。

## 一、考察重点

1. 数据探索作用：面对新领域或缺乏业务背景的数据时，初步了解数据内容、性质、质量，评估完整性和一致性，识别数据模式和分布，为后续预处理与分析技术选择提供依据。
2. 数据属性类型：定量数据包括连续型和离散型；定性数据包括标称型/定类和有序型/定序。
3. 集中量数：均值、截断均值、中位数、众数、百分位数；均值受极端值影响大，中位数和众数相对不受极端值影响。
4. 差异量数：极差、标准差、方差、中位数绝对偏差MAD、四分位差IQR；MAD对异常值极度不敏感，IQR代表中间50%数据波动范围。
5. 相关性量数：协方差衡量两个变量变化趋势一致性，同向为正、反向为负；同一变量的协方差就是方差；Pearson相关系数衡量线性相关程度；协方差矩阵对角线为方差，非对角线为协方差。

## 二、知识点分析

### 知识点一：数据探索的作用

- 数据探索用于面对新领域、新行业或缺乏业务背景的数据时，初步了解数据内容、性质和质量。
- 核心作用：评估数据完整性和一致性；识别数据性质、模式和分布；为后续选择预处理工具和分析技术提供依据。
- 考法提示：如果问“为什么建模前要探索数据”，要回答它帮助理解数据质量和分布，而不是直接建模。

-数据探索的实际应用场景包括：

1. 客户分群与画像 — 通过探索用户属性（年龄、消费金额、活跃时长）的分布和相关性，发现自然聚类，为后续RFM分析或K-Means聚类提供依据
2. 风控与异常检测 — 利用IQR、箱线图探索交易数据的离群点，识别欺诈行为或设备异常
3. 特征工程前的数据诊断 — 探索缺失率、方差、分布偏态，决定是否需要填充、标准化或做对数变换
4. A/B实验效果分析 — 用描述性统计和可视化对比实验组与对照组的集中趋势和离散程度，判断指标差异的显著性
5. 数据质量评估 — 面对新接入的业务表（如CRM、ERP），先用探索性分析检查字段完整性、一致性、是否有多模态分布，再决定清洗策略
   核心价值在于：*先摸清数据"长什么样"，再决定"怎么建模"*，避免盲目套用算法导致结论不可靠。

* 可视化(Visualization)：将数据转化为可视的图或表格展现。人的大脑对图形化的数据模式认知能力，是任何纯数字化分析工具都无法比拟的。
* 数据统计分析：利用统计指标对数据流的分布特性进行定量描述。
* | 作用     | 可视化表现                                   | 纯数据表现                             |
  | -------- | -------------------------------------------- | -------------------------------------- |
  | 模式认知 | 形状、颜色和聚类立刻揭示数据结构（如相关性） | 需复杂统计计算才能发现规律             |
  | 趋势评估 | 坡度陡峭程度直接反映增长或下降速度           | 必须对比前后数值，难以直观感受变化速率 |
  | 异常检测 | 远离群体的点非常"刺眼"，一眼可见             | 异常值淹没在大量数据中，极易被忽略     |
* 人类视觉优势：以视觉角度分析海量信息的能力，尤其在模式认知、变化趋势评估、异常值检测和异常模式识别上。

### 知识点二：数据属性类型

- 定量数据：连续型和离散型。连续型可取区间内任意值，如销售额、身高、温度；离散型只能取特定孤立值，如企业个数、职工人数。
- 定性数据：标称型和有序型。-标称型无顺序差异，如性别、血型；有序型有顺序但不能常规加减，如成绩等级、满意度等级。
- 易错点：有序型虽然有顺序，但不能当作连续数值随意做数学运算。

### 知识点三：集中量数

- 均值：数据总和除以数据个数，最常用但受极端值影响大。
- 截断均值：去掉两端最高和最低的一定比例后再求均值，用于削弱极端值影响。
- 中位数：排序后中间位置的值，不易受极端值影响。
- 众数：出现次数最多的值，也不直接受极端值大小影响，但可能不唯一或不存在。
- 百分位数：用于表示数据中某个位置以下数据所占比例。

- | 量数     | 定义                                  | 特点                                           |
  | -------- | ------------------------------------- | ---------------------------------------------- |
  | 频率     | 某数值出现的次数/比例                 | 基础描述统计                                   |
  | 均值     | 数据总和/数据个数                     | 受极端值影响大；可用**截断**均值改善     |
  | 中位数   | 排序后中间位置的值                    | 不受极端值影响                                 |
  | 众数     | 出现次数最多的值                      | 不受极端值影响；可能不唯一或不存在；缺乏敏感性 |
  | 百分位数 | 有的数据<该值；分组数据需用组下限计算 | 地位量数                                       |
- **截断**：先把所有数据从小到大排序，然后去掉两头最高和最低的一定比例（比如去掉最高5%和最低5%）的数据，只用中间剩下的“温和数据”来计算平均值。

### 知识点四：差异量数与稳健统计量

- 极差/全距：最大值减最小值，简单但极易受极端值影响。
- 方差与标准差：反映数据离散程度，常用但也会受离群点影响。
- MAD：中位数绝对偏差，各数据与中位数的绝对差的中位数，对异常值极度不敏感，是材料中强调的稳健差异量数。
- IQR：四分位极差，等于Q3-Q1，代表中间50%数据的波动范围，抗异常值能力极强。


### 知识点五：协方差、相关系数与矩阵

- 协方差：衡量两个变量变化趋势是否一致。同向变化为正，反向变化为负；同一变量的协方差就是方差。
- Pearson相关系数：衡量两个随机变量之间线性相关程度。
- 协方差矩阵：对角线元素是方差，非对角线元素是协方差。
- 相关矩阵：第i个和第j个属性之间的相关性，对角线为1。
- 注意：相关关系不等于因果关系，材料只要求掌握相关程度和正负含义。
- 协方差：衡量两个变量变化趋势的一致性（同向为正，反向为负）：方差是协方差的特例
  （同一变量）。
- 相关系数（Pearson）：衡量两个随机变量X、Y之间线性相关程度的指标，取值[-1，1]；
  r=0.994表示高度线性正相关。
- 协方差矩阵：对称矩阵，对角线为方差，非对角线为协方差。
- 相关矩阵：第ij个元素是第i个和第j个属性之间的相关性，对角线为1。
- 协方差（Covariance）的物理意义

  - 定义：用于衡量两个变量之间的总体误差，反映两个变量的变化趋势是否一致。
  - 物理意义：
    - 若两个变量变化趋势一致（一个大于期望，另一个也大于期望），则协方差为正值。
    - 若两个变量变化趋势相反，则协方差为负值。
    - 当两个变量是同一个变量时，其协方差就是该变量的方差。

## 三、复习步骤

1. 先按数据类型背例子：性别/血型是标称，成绩等级/满意度是有序，销售额是连续，企业个数是离散。
2. 再按统计量分类：集中量数回答“中心在哪里”，差异量数回答“波动有多大”，相关性量数回答“两个变量如何一起变”。
3. 最后练解释题：看到极端值优先想到均值、极差、标准差受影响；看到稳健统计量优先想到中位数、MAD、IQR。

## 四、易错点

- 有序型虽然有顺序，但不能直接做普通加减运算。
- 相关系数强调线性相关，不等同于因果关系；材料只要求掌握相关程度与协方差含义。

## 六、例题

### 1.说明数据探索在数据挖掘流程中的作用。

1. 初步了解数据 — 面对新领域或缺乏业务背景的数据时，摸清数据内容、性质和质量
2. 评估完整性与一致性 — 检查缺失值、异常值、格式是否统一
3. 识别数据模式和分布 — 通过可视化与统计量发现数据的集中趋势、离散程度、相关关系
4. 为后续预处理和分析技术选择提供依据 — 根据探索结果决定是否要标准化、填补缺失值、做特征变换，以及选择什么样的建模算法
   一句话总结：*先摸清数据"长什么样"，再决定"怎么建模"*，避免盲目套用算法。
