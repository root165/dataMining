📍 KNN k-近邻

训练阶段
主要保存全部训练样本，不建模型 → 惰性学习
预测阶段
1. 计算未知样本与所有训练样本的距离（如欧氏距离）
2. 选出距离最近的 k 个邻居
3. 这 k 个邻居进行多数投票，得票最多的类别为预测结果
关键问题：
⚠ 对量纲敏感——"收入(万)"和"年龄(岁)"数值差异大，年龄的变化会被淹没。
✅ 解决方法：标准化（Z-score）或归一化（Min-Max）
⚠ k 值选择：k 太小 → 过拟合（噪声敏感）；k 太大 → 欠拟合（类别边界模糊）
KNN 分类示意（k=3）
★ 未知点 · 3 个最近邻：2 个蓝 / 1 个红 → 预测为蓝色类

📈 逻辑回归 Logistic Regression

名字带"回归"，但主要用于分类！
核心思想
在线性回归的基础上套一层 Sigmoid 函数，将输出压缩到 (0,1) 区间，
表示样本属于正类的概率。
P(y=1|x) = σ(w·x + b) = 1 / (1 + e-(w·x + b))
决策规则：P ≥ 0.5 → 正类；P < 0.5 → 负类
损失函数：交叉熵损失（并非 MSE）
输出：可解释为概率（如：贷款违约概率 85%）
Sigmoid 函数曲线
z < 0 → P < 0.5（负类）；z > 0 → P > 0.5（正类）

🔲 SVM 支持向量机

核心思想
寻找一个最大间隔超平面，使两类样本被分开的同时，距离超平面最近的样本点（支持向量）到超平面的距离最大。
支持向量（Support Vectors）
离超平面最近的那几个训练样本，唯一决定分类边界。
其他远离边界的样本不影响分类面。
核函数（Kernel Trick）
将数据映射到高维空间，使原本线性不可分的数据变得线性可分。
常见核函数：线性核、多项式核、RBF（高斯）核
主要作用：处理非线性问题
SVM 最大间隔超平面
实线 = 决策边界 · 虚线 = 间隔边界 · ⭕ 上的点 = 支持向量

前向传播：ŷ = σ(W₂·σ(W₁·x + b₁) + b₂)
损失函数：交叉熵（分类）/ MSE（回归）
反向传播：链式法则求损失对各层权重的梯度
优化器：梯度下降（SGD、Adam 等）更新参数
前向传播 = 预测 · 反向传播 = 学习
梯度下降直观理解
想象你在山顶（误差很大），要走到山谷（误差最小）。
梯度 = 当前最陡的下山方向
学习率 η = 每一步迈多大
走得太小（η 太小）→ 太慢；走得太快（η 太大）→ 跳过山谷

🔒 正则化 Regularization

目的：限制模型复杂度，缓解过拟合
L1 正则化（Lasso）
Loss + λ Σ|wi|
✅ 使部分权重变为 0
✅ 自带特征选择
✅ 产生稀疏模型
L2 正则化（Ridge）
Loss + λ Σwi2
✅ 使权重缩小但不为零
✅ 对异常值更稳健
✅ 岭回归使用 L2
弹性网络（ElasticNet）
L1 + L2 混合
✅ 结合两者优点
✅ 适合高相关特征
过拟合表现
训练误差很低甚至为 0
但测试/检验误差很高
模型记住了噪声而非规律
处理过拟合的方法
① 增加训练数据
② 正则化（L1/L2）
③ 剪枝（决策树）
④ 交叉验证
⑤ 早停（Early Stopping）
⑥ 降低模型复杂度

📋 五大算法横向对比

算法核心原理优点缺点关键注意
KNN 距离 + 多数投票 简单、无训练 预测慢、维度灾难 需标准化
逻辑回归 Sigmoid 输出概率 可解释性强 只能线性决策边界 用于分类！
SVM 最大间隔超平面 核函数处理非线性 大数据集训练慢 支持向量决定边界
神经网络 多层 + 反向传播 拟合能力强 需要大量数据 小心过拟合
正则化 在损失中加惩罚项 防止过拟合 需调 λ 参数 L1 做特征选择
