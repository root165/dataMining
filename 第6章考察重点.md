## 一、考察重点

1. 分类定义：将数据对象映射到预定义类别中；训练样本应包含属性值和类别标记，模型训练后用于未知样本分类。
2. 分类过程：选择训练集、选择测试集、建立模型、测试评估、应用模型；可通过交叉验证轮换选择模型。
3. 决策树结构：内部节点表示属性测试，叶节点存放类标号；对未知样本沿根到叶路径得到预测类别。
4. 属性选择与不纯度：划分后应使不纯度降低；信息增益选择降低熵最多的属性，但偏向取值较多的属性；信息增益率用于惩罚分支多的属性；Gini指数选择Gini split最小者。
5. 决策树剪枝：预剪枝在树生长过程中提前停止，速度快但可能过早；后剪枝先完整生长再修剪，效果较好但计算量大。
6. 过拟合原因与表现：噪声或缺乏代表性样本会导致决策边界改变；训练误差很低但检验/测试误差很高。
7. 朴素贝叶斯：基于贝叶斯公式和条件独立假设；区分先验概率、似然、后验概率；优点是计算快、规则简单、对噪声较鲁棒，局限是独立性假设实践中不一定成立。
8. 期末复习大纲额外高频：分类/回归区别、KNN距离与量纲敏感、逻辑回归用于分类、SVM最大间隔与核函数、神经网络前向传播/损失函数/反向传播/优化器、L1/L2正则化缓解过拟合。

## 二、知识点分析

### 知识点一：分类的定义与一般过程

- 分类就是把数据对象映射到预定义类别中，也可以理解为“贴标签”。
- 训练样本需要包含属性值和类别标记，用于构建分类模型。
- 一般过程：选择训练数据集→选择测试数据集→建立模型→测试评估→应用模型。
- 交叉验证：可通过互换或轮换训练/测试数据选择更优模型。

### 知识点二：决策树分类原理

- 决策树分类利用树结构构造分类模型，发现数据中蕴含的分类规则。
- 内部节点：表示在某个属性上的测试。叶节点：存放类标号。
- 分类方式：对未知类别样本，从根节点沿测试路径走到叶节点，叶节点类别即预测类别。
- 考法提示：常考“内部节点是什么、叶节点是什么、未知样本如何分类”。

### 知识点三：不纯度度量与属性选择

- 核心思想：划分后应使子节点不纯度降低，即某一类占据更明显优势。
- 信息增益：划分前熵减去划分后加权平均熵，选择信息增益最大的属性。
- 信息增益缺点：偏向取值较多、分支较多的属性（如身份证号）。
- 信息增益率：对分支多的属性进行惩罚，是对信息增益偏向的修正。
- Gini指数：选择Gini split最小的划分。

核心思想：划分后应使不纯度减小，即某一类占绝对优势。

### 预备概念：熵（Entropy）

熵用来度量"混乱程度"。想象一个袋子里有红球和蓝球：

- 如果全是红球 → 非常确定 → 熵 = 0
- 如果一半红一半蓝 → 最不确定 → 熵 = 1

**公式**：熵 = - (概率1 × log₂概率1 + 概率2 × log₂概率2 + ...)

**举例**：10 个样本，5 个买电脑、5 个不买。

熵 = -(0.5×log₂0.5 + 0.5×log₂0.5) = -(0.5×(-1) + 0.5×(-1)) = 1

### 四种度量标准

---

#### ① 信息增益（InfoGain）

**思路**：用某个属性划分后，熵降低了多少。降得越多越好。

**公式**：信息增益 = 划分前的熵 - 划分后各分支的熵（加权平均）

**举例**：按"年龄"划分买电脑数据。

| 年龄 | 买 | 不买 | 共 |
| ---- | -- | ---- | -- |
| 青年 | 2  | 3    | 5  |
| 中年 | 4  | 0    | 4  |
| 老年 | 3  | 2    | 5  |

划分前熵 = 1（前面算过）。

- 青年分支：熵 = -(2/5×log₂(2/5) + 3/5×log₂(3/5)) ≈ 0.97
- 中年分支：全是买 → 熵 = 0
- 老年分支：熵 = -(3/5×log₂(3/5) + 2/5×log₂(2/5)) ≈ 0.97

划分后加权熵 = (5/14)×0.97 + (4/14)×0 + (5/14)×0.97 ≈ 0.69

信息增益 = 1 - 0.69 = **0.31**

**特点**：好用，但偏向选择取值多的属性（如"身份证号"每人一个值，信息增益极大，但分类毫无意义）。

---

#### ② 信息增益率（GainRatio）

**思路**：在信息增益基础上，惩罚分支太多的属性。

**公式**：增益率 = 信息增益 ÷ 分裂信息

分裂信息 = 对"属性分了多少支"的惩罚项，分支越多数值越大。

**举例**：上面"年龄"分了 3 支。

分裂信息 = -(5/14×log₂(5/14) + 4/14×log₂(4/14) + 5/14×log₂(5/14)) ≈ 1.58

增益率 = 0.31 ÷ 1.58 ≈ **0.20**

如果拿"身份证号"划分（每人一支），分支极多，分裂信息会很大，增益率被拉低，所以不会被误选。

**特点**：修正了信息增益的缺陷。

---

#### ③ Gini 系数

**思路**：随机抽两个样本，它们属于不同类别的概率。这个概率越小越好。

**公式**：Gini = 1 - (类别A概率)² - (类别B概率)² - ...

**举例**：10 个样本，5 买 5 不买。

Gini = 1 - (0.5)² - (0.5)² = 1 - 0.25 - 0.25 = **0.5**

如果 9 买 1 不买：

Gini = 1 - (0.9)² - (0.1)² = 1 - 0.81 - 0.01 = **0.18**

**特点**：计算比熵快（不用算 log），CART 决策树在用。

---

#### ④ 分类误差

**思路**：最简单的办法——看哪类占比最大，错误率就是 1 减去这个占比。

**公式**：分类误差 = 1 - 占比最大的类别的概率

**举例**：10 个样本，5 买 5 不买 → 最大占比 = 0.5 → 误差 = 1 - 0.5 = **0.5**

9 买 1 不买 → 最大占比 = 0.9 → 误差 = 1 - 0.9 = **0.1**

**特点**：简单直观，但对不纯度的变化不够敏感（比如从 8:2 到 9:1，Gini 和熵都有明显变化，但分类误差变化很小）。

---

### 总结对比

| 度量标准   | 一句话概括                       | 选择标准 |
| ---------- | -------------------------------- | -------- |
| 信息增益   | 划分后混乱度降了多少             | 越大越好 |
| 信息增益率 | 降混乱度的同时扣掉分支太多的罚分 | 越大越好 |
| Gini 系数  | 随机抽两个是不同类别的概率       | 越小越好 |
| 分类误差   | 猜错的比例                       | 越小越好 |

### 知识点四：决策树剪枝与过拟合

- 剪枝目的：防止过拟合，提高模型在测试数据上的泛化能力。
- 预剪枝：在树生长过程中提前停止。优点是速度快，缺点是可能剪枝过早。
- 后剪枝：先让树完整生长，再修剪不必要分支。优点是效果较好，缺点是计算量较大。
- 过拟合表现：训练误差很低甚至为0，但检验/测试误差很高。
- 过拟合原因：噪声会改变决策边界，缺乏代表性样本会导致模型根据少量记录做出错误决策。

### 知识点五：朴素贝叶斯分类

- 先验概率：不看任何特征时，某类出现的概率。
- 似然：在某类中出现这些特征的概率。
- 后验概率：看到特征后，某类出现的概率。
- 朴素贝叶斯假设：所有特征条件独立，用于简化计算。
- 优点：计算快、规则简单、对噪声较鲁棒、容易处理缺失值。局限：实践中特征独立性假设往往不完全成立。

| 概念                             | 解释                             |
| -------------------------------- | -------------------------------- |
| 先验概率$P(C)$                 | 不看任何特征时，某类出现的概率   |
| 似然$P(\mathbf{X} \mid C)$     | 某类中，出现这些特征的概率       |
| 后验概率$P(C \mid \mathbf{X})$ | 看到特征后，某类出现的概率       |
| 朴素贝叶斯                       | 假设所有特征互相独立（简化计算） |

### 核心公式

贝叶斯定理：

$$
P(C \mid \mathbf{X}) = \frac{P(\mathbf{X} \mid C) \, P(C)}{P(\mathbf{X})}
$$

分类决策时只比较分子，分母相同可忽略：

$$
P(C \mid \mathbf{X}) \propto P(\mathbf{X} \mid C) \times P(C)
$$

朴素贝叶斯的独立性假设：

$$
P(\mathbf{X} \mid C) = \prod_{i=1}^{n} P(X_i \mid C)
$$

因此朴素贝叶斯分类器选择后验概率最大的类别：

$$
\hat{C} = \arg\max_{C} \, P(C) \prod_{i=1}^{n} P(X_i \mid C)
$$

### 理论性质

朴素贝叶斯分类器理论上具有最小的错误率，但在实践中由于特征独立性假设往往不成立，所以不一定最优。

| 优点             | 缺点                 |
| ---------------- | -------------------- |
| 计算快，规则简单 | 要求特征独立性假设   |
| 对噪声鲁棒       | 属性相关时会降低效果 |
| 容易处理缺失值   | 需要概率数据         |
| 理论上错误率最小 | 实践中不一定最优     |

### 知识点六：分类预测客观题高频算法

- KNN：基于距离和多数投票，训练阶段主要保存样本，属于惰性学习；对量纲敏感，常需标准化。
- 逻辑回归：名字带回归，但主要用于分类，输出类别概率。
- SVM：寻找最大间隔超平面，支持向量决定分类边界，核函数用于处理非线性问题。
- 神经网络：前向传播得到预测，损失函数度量误差，反向传播计算梯度，优化器更新参数。
- 正则化：L1/L2用于限制模型复杂度、缓解过拟合；岭回归使用L2正则。

## 三、复习步骤

1. 先抓分类主线：有标签训练-建模-测试-应用，这是和聚类的根本差异。
2. 决策树重点背结构、分类路径、信息增益偏向、增益率原因、剪枝作用。
3. 再用表格横向记算法：KNN看距离且要标准化；朴素贝叶斯看概率和独立假设；SVM看最大间隔和核函数；神经网络看前向、损失、反向、更新。
4. 最后把过拟合与正则化、剪枝、交叉验证连起来答。

## 四、易错点

- 信息增益不是完美指标，它偏向取值多的属性。
- 逻辑回归名字带回归，但复习大纲要求记住它主要用于分类。
- KNN对量纲敏感，使用距离前常需标准化。
